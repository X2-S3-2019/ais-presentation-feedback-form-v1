Student Assessment Software

Welcome to Student Assessment Software

To start using student presentations software, you should do the following actions:

Press "Setup Presentations" button and create courses with course names and ids.
Once the courses have been set, click the Presentations tab and select a course and create a presentation with name and date of the presentation.

When courses and presentations are set, press "Set files path" button to setup the saving path for word files.

Next, press click on "Go to assessment" button to start the presentation assessment.
Select an appropriate course and presentation. Input any student ID and press "Get student full name" button to get the student name by his id.
You can click "Start Presentations Timer" button to start the counter of the presentation length.
Click on tiles in the assessment table to make an assessment. When the assessment is finished, click Save button to get assessment as a word file.
