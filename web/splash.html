How to Use the Presentation Assessment Software


To start using student presentations software, you should do the following:

Press the "Setup Presentations" button to create courses by inputting the course ID and name.

Once the courses have been set, click the Presentations tab and select a course. Create a presentation by inputting the date of the presentation and the topic.

When courses and presentations are set, press "Set files path" button to specify where the generated word documents will be saved.

Click on the "Assessment" button to start evaluating presentations.

Input the student ID and press "Get student name" button to get the student name. Then, select the appropriate course and presentation.

You can click "Start Presentations Timer" button to keep track of the presentation's duration.

Click on the appropriate assessment based on the criteria. When the assessment is finished, click "Review" button to review the assessment scores.

Once you've reviewed the assessment, click the "Save" button to generate the word document.
